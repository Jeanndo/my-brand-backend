# MY-BRAND REST APIs

### LIST OF APIs FOR USERS AUTHENTICATION

`

- POST/api/v1/users/signup
- POST/api/v1/users/login
- POST/api/v1/users/forgotPassword
- GET/api/v1/users/
- GET/api/v1/users/:id
- PATCH/api/v1/users/:id
- DELETE/api/v1/users/:id

`
